Chatbot

Basic Chatbot is a simple, front-end project developed using HTML, CSS, and JavaScript to simulate basic conversational interactions. Designed with a clean and user-friendly interface, the chatbot responds to user inputs with predefined replies, demonstrating foundational concepts of DOM manipulation, event handling, and conditional logic. Created at the end of my first year during an internship, this project helped strengthen my understanding of interactive web development and laid the groundwork for more advanced projects in the future.
