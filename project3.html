Project Charity

Built using HTML, CSS, and JavaScript, the project features a visually engaging design with interactive elements to simulate a donation platform. Although non-functional, the project effectively demonstrates front-end development skills, user interface design, and the ability to convey important messages about environmental responsibility through digital media. This project was developed during an internship at the end of my first year of engineering, providing valuable hands-on experience in collaborative web development. Project Charity is a group project focused on creating an interactive web page to promote environmental awareness and encourage donations for sustainability causes.
